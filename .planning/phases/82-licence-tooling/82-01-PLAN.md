Key generation (PKCS8, NoEncryption):
```python
from cryptography.hazmat.primitives import serialization
from cryptography.hazmat.primitives.asymmetric.ed25519 import Ed25519PrivateKey

private_key = Ed25519PrivateKey.generate()
private_pem = private_key.private_bytes(
    encoding=serialization.Encoding.PEM,
    format=serialization.PrivateFormat.PKCS8,
    encryption_algorithm=serialization.NoEncryption(),
)
public_pem = private_key.public_key().public_bytes(
    encoding=serialization.Encoding.PEM,
    format=serialization.PublicFormat.SubjectPublicKeyInfo,
)
```

Write `axiom-licenses/tools/issue_licence.py` — an evolution of `tools/generate_licence.py` with three additions: YAML audit records, GitHub API commit, and strict key-source resolution.

CLI args:
- `--customer` (required) — customer identifier, e.g. acme-corp

- `--nodes` (required, type=int) — max active nodes (0 = unlimited)

- `--contact-email` (optional, default="") — contact email
- `--grace-days` (optional, type=int, default=30) — grace period in days
- `--key` (optional) — path to private key PEM file
- `--issued-by` (optional) — defaults to git config user.name if available, else getpass.getuser()
- `--no-remote` (flag) — write YAML locally and print JWT; skip GitHub API

Key resolution (`resolve_key` function):
```python
import os, sys, getpass, subprocess
from pathlib import Path
from cryptography.hazmat.primitives import serialization

def resolve_key(args):
    key_source = args.key or os.getenv("AXIOM_LICENCE_SIGNING_KEY")
    if not key_source:
        sys.exit(
            "Error: no signing key provided.\n"
            "Set AXIOM_LICENCE_SIGNING_KEY env var or pass --key <path>."
        )
    path = Path(key_source)
    if not path.exists():
        sys.exit(f"Error: key file not found: {path}")
    return serialization.load_pem_private_key(path.read_bytes(), password=None)
```

`resolve_issued_by(args)` helper: if args.issued_by is set, use it. Otherwise try `subprocess.check_output(["git","config","user.name"], text=True).strip()`, fall back to `getpass.getuser()` if subprocess fails.

`build_audit_record(payload, token, issued_by)`: returns dict with all 12 YAML fields. The `jti` field is set from `payload["licence_id"]`. Use `datetime.datetime.utcfromtimestamp(payload["iat"]).date().isoformat()` for `issued_at`.

`commit_yaml_to_github(customer_id, jti, yaml_text, tier, expiry, github_token)`: uses `requests.put` to the GitHub Contents API. If `AXIOM_GITHUB_TOKEN` is not set, exit with: "Error: AXIOM_GITHUB_TOKEN env var not set. Use --no-remote for local-only operation."

`main()`:
1. Parse args
2. resolve_key
3. Sign JWT (same pattern as generate_licence.py but with new arg names)
4. build_audit_record
5. Serialise with `yaml.dump(record, default_flow_style=False, sort_keys=False)`
6. If `--no-remote`: write `{customer_id}-{jti}.yml` to current dir, print YAML to stderr, print JWT to stdout
7. Else: call commit_yaml_to_github; print JWT to stdout; print commit confirmation to stderr

Print a human-readable summary to stderr (same pattern as generate_licence.py) showing all fields.
