**Purpose:** Notify all connected dashboard clients when licence status changes.

**Definition of Done:**
- [ ] In `puppeteer/agent_service/main.py`:
  - Add WebSocket event type to broadcast pool:
    ```python
    # In the reload_licence_endpoint (after state swap):

        "old_status": str(old_state.status),

        "timestamp": datetime.utcnow().isoformat()
    })
    ```

### Task 2: Extend useWebSocket.ts Hook for Licence Events

**Purpose:** Allow React components to subscribe to and handle licence status changes.

**Definition of Done:**
- [ ] In `puppeteer/dashboard/src/hooks/useWebSocket.ts`:
  - Add event handler for `licence_status_changed`:
    ```typescript
    const handleLicenceStatusChanged = (data: {
        old_status: string;
        new_status: string;
        message: string;
        timestamp: string;
        metadata?: {
            organization: string;
            tier: string;
            expires_at: string;
        };
    }) => {
        // Call optional callback provided by components
        if (onLicenceStatusChanged) {
            onLicenceStatusChanged(data);
        }
        // Optionally: refresh Admin page licence section
    };
    ```
  - Expose `onLicenceStatusChanged` callback in hook return:
    ```typescript
    return {
        isConnected,
        onLicenceStatusChanged,  // optional callback setter
        ...
    };
    ```
  - Pattern: components that need licence updates call `useWebSocket()` and subscribe to this event
- [ ] No UI changes in this task (just hook infrastructure)
- [ ] Test:
  - Subscribe to licence event in a test component
  - Verify callback fires when broadcast received
  - Verify data structure matches backend payload

**Resources:**
