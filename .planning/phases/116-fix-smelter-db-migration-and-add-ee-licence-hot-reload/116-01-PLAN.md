### Task 5: Add Background Licence Expiry Timer

**Purpose:** Periodically check licence expiry and update state without requiring manual reload.

**Definition of Done:**
- [ ] In `puppeteer/agent_service/main.py` lifespan:
  - Create async background task in `lifespan` event:
    ```python
    async def licence_expiry_checker():
        while True:
            try:
                await asyncio.sleep(60)  # check every 60s
                if not app.state.licence_state:
                    continue
                new_status = check_licence_expiry(app.state.licence_state)
                if new_status != app.state.licence_state.status:
                    old_status = app.state.licence_state.status
                    app.state.licence_state.status = new_status
                    # Wave 2: broadcast_ws_event("licence_status_changed", ...)
                    # Wave 2: audit("licence_status_changed", ...)
            except Exception as e:
                logger.error(f"Licence expiry check failed: {e}")
    ```
  - Create task in lifespan: `asyncio.create_task(licence_expiry_checker())`
  - Ensure task cancels on shutdown (lifespan context manager cleanup)
- [ ] Test:

  - No errors on timer cancel during shutdown

**Resources:**

