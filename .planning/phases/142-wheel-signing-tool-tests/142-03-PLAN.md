**Pattern 1: Keypair generation**

**Pattern 2: No-overwrite protection**
- Write dummy content to out_path: out_path.write_bytes(b"existing")
- Call generate_keypair(out_path, force=False)
- Expect pytest.raises(SystemExit) with message containing "already exists"

**Pattern 3: Public key bytes literal format**
- Call generate_keypair() to get (private_pem, public_pem)
- Import and call the main() entry point? OR capture stdout from generate_keypair return?

- Option A: Call main() via sys.argv mocking (complex)
- Option B: Test generate_keypair() return directly, verify public_pem is bytes with PEM format
- For "bytes literal" test: verify that public_pem decoded looks like `b"""-----BEGIN..."""` when printed
- Use capsys.readouterr() if calling a printing function, or just verify the bytes format

print(f'b"""{public_pem.decode()}"""')

**Pattern 4: Force flag behavior**
- Write existing content: existing_path.write_bytes(b"old key")

**Pattern 5: File permissions**

Test implementation notes:

test_generate_keypair:
- out_path = temp_wheel_dir / "test.key"
- private_pem, public_pem = generate_keypair(out_path, force=False)
- Assert isinstance(private_pem, bytes)
- Assert isinstance(public_pem, bytes)
- Assert out_path.exists()
- Assert b"-----BEGIN PRIVATE KEY-----" in private_pem
- Assert b"-----BEGIN PUBLIC KEY-----" in public_pem
- Assert out_path.read_bytes() == private_pem

test_no_overwrite_without_force:
- out_path = temp_wheel_dir / "test.key"
- out_path.write_bytes(b"existing key\n")
- with pytest.raises(SystemExit) as exc_info:
-     generate_keypair(out_path, force=False)
- Assert "already exists" in str(exc_info.value).lower()

test_public_key_bytes_literal:
- out_path = temp_wheel_dir / "test.key"
- private_pem, public_pem = generate_keypair(out_path, force=False)
- formatted = f'b"""{public_pem.decode()}"""'
- Assert formatted.startswith('b"""')
- Assert "-----BEGIN PUBLIC KEY-----" in formatted
- Assert formatted.endswith('"""')

test_force_flag_overwrites:
- out_path = temp_wheel_dir / "test.key"
- old_content = b"old key content\n"
- out_path.write_bytes(old_content)
- private_pem, public_pem = generate_keypair(out_path, force=True)
- new_content = out_path.read_bytes()
- Assert new_content != old_content
- Assert b"-----BEGIN PRIVATE KEY-----" in new_content

test_file_permissions_0600:
- out_path = temp_wheel_dir / "test.key"
- private_pem, public_pem = generate_keypair(out_path, force=False)
- stat = out_path.stat()
- mode = stat.st_mode & 0o777
- Assert mode == 0o600  # Readable/writable only by owner

Alternative for permissions test (more portable):

import stat

But simpler approach: `(stat.st_mode & 0o777) == 0o600`
