From Wave 1 (114-01):

**Bundle endpoints:**
- GET /api/admin/bundles → list[CuratedBundleResponse]
- POST /api/admin/bundles → CuratedBundleResponse
- PATCH /api/admin/bundles/{id} → CuratedBundleResponse
- DELETE /api/admin/bundles/{id} → {status, id}
- POST /api/admin/bundles/{id}/items → CuratedBundleItemResponse
- DELETE /api/admin/bundles/{id}/items/{item_id} → {status, item_id}

**CuratedBundleResponse type:**
```typescript
interface CuratedBundleResponse {
  id: string;
  name: string;
  description?: string;
  ecosystem: "PYPI" | "APT" | "APK" | "CONDA" | "NUGET" | "OCI" | "NPM";
  os_family: "DEBIAN" | "ALPINE" | "WINDOWS";
  is_active: boolean;
  created_at: string;
  items: CuratedBundleItemResponse[];
}

interface CuratedBundleItemResponse {
  id: string;
  bundle_id: string;
  ingredient_name: string;
  version_constraint: string;
  ecosystem: string;
}
```

**PuppetTemplate interface:**
```typescript
interface Template {
  id: string;
  friendly_name: string;
  is_starter: boolean;
  status: "ACTIVE" | "DEPRECATED" | "REVOKED";
  base_image?: string;
  package_count?: number;
  description?: string;
}
```
