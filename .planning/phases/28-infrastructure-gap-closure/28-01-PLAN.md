**Step 2 — Verify CDN-free output (required):**
```bash

  sh -c "grep -rq 'fonts.googleapis.com\|cdn.jsdelivr.net\|cdnjs.cloudflare.com' /usr/share/nginx/html \
         && echo FAIL || echo PASS"
```
This must print `PASS`. Polarity: grep exits 0 on a match (found = bad = FAIL), and the `|| echo PASS` branch runs when grep exits non-zero (nothing found = good = PASS). Both build success AND `PASS` from this grep are required to close INFRA-06.
