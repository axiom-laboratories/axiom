# Phase 168 Plan 02 — EE Gating and Router Setup

## Goal
Implement EE gating for SIEM endpoints and create the admin API routes (GET/PATCH config, test-connection, status). Integrate SIEMService into the application lifespan so it initializes on startup and is accessible to routers. Add SIEM status to the system health endpoint.

## Context
This plan wires SIEMService (created in Plan 01) into the FastAPI application. It establishes the CE/EE boundary via the stub router pattern (inherited from Phase 167), registers the EE admin endpoints, and ensures the service is instantiated and available for the deps.py audit() hook (Plan 03). Plan 02 also adds the siem status field to GET /system/health alongside the existing vault status.

## Threat Model

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-168-06 | Elevation of Privilege | Admin endpoints (/admin/siem/*) | mitigate | All endpoints require `require_ee()` gate + `require_permission("config:write")` per D-01; CE users get 402 immediately |
| T-168-07 | Information Disclosure | Status endpoint leaks config details | mitigate | `GET /admin/siem/status` only returns error_detail if user is admin (checked via permission); degraded error messages are generic |
| T-168-08 | Denial of Service | Test-connection endpoint hangs | mitigate | `_test_connection()` in SIEMService has 10s timeout via httpx; syslog test uses 2s socket timeout; no user-supplied DNS resolution |

**Endpoint 1: GET /admin/siem/config**
```python
@router.get("/config", response_model=SIEMConfigResponse)
async def get_config(
    db: AsyncSession = Depends(get_db),
    _=Depends(require_permission("config:write")),
):
    """Retrieve current SIEM configuration."""
    result = await db.execute(select(SIEMConfig).limit(1))
    config = result.scalars().first()
    if not config:
        raise HTTPException(status_code=404, detail="SIEM config not found")
    return SIEMConfigResponse.from_siem_config(config)
```

**Endpoint 2: PATCH /admin/siem/config**
```python
@router.patch("/config", response_model=SIEMConfigResponse)
async def update_config(
    req: SIEMConfigUpdateRequest,
    db: AsyncSession = Depends(get_db),
    _=Depends(require_permission("config:write")),
):
    """Update SIEM configuration. Reinitialize service if enabled changed."""
    result = await db.execute(select(SIEMConfig).limit(1))
    config = result.scalars().first()
    if not config:
        raise HTTPException(status_code=404, detail="SIEM config not found")
    
    # Update only provided fields
    if req.backend is not None:
        config.backend = req.backend
    if req.destination is not None:
        config.destination = req.destination
    if req.syslog_port is not None:
        config.syslog_port = req.syslog_port
    if req.syslog_protocol is not None:
        config.syslog_protocol = req.syslog_protocol
    if req.cef_device_vendor is not None:
        config.cef_device_vendor = req.cef_device_vendor
    if req.cef_device_product is not None:
        config.cef_device_product = req.cef_device_product
    if req.enabled is not None:
        config.enabled = req.enabled
    
    await db.commit()
    await db.refresh(config)
    
    # Reinitialize service to pick up new config (will be done in Plan 3 integration)
    return SIEMConfigResponse.from_siem_config(config)
```

**Endpoint 3: POST /admin/siem/test-connection**
```python
@router.post("/test-connection", response_model=SIEMTestConnectionResponse)
async def test_connection(
    req: SIEMTestConnectionRequest,
    _=Depends(require_permission("config:write")),
):
    """Test connectivity to the configured SIEM destination."""
    siem = get_siem_service()
    if not siem:
        return SIEMTestConnectionResponse(
            success=False,
            status="disabled",
            message="SIEM service not initialized"
        )
    
    try:
        await siem._test_connection()
        return SIEMTestConnectionResponse(
            success=True,
            status="healthy",
            message="Successfully connected to SIEM destination"
        )
    except Exception as e:
        return SIEMTestConnectionResponse(
            success=False,
            status="degraded",
            error_detail=str(e),
            message=f"Connection failed: {str(e)}"
        )
```

**Endpoint 4: GET /admin/siem/status**
```python
@router.get("/status", response_model=SIEMStatusResponse)
async def get_status(
    _=Depends(require_permission("config:read")),
):
    """Retrieve SIEM service status."""
    siem = get_siem_service()
    if not siem:
        return SIEMStatusResponse(status="disabled")
    
    return SIEMStatusResponse(
        status=await siem.status(),
        backend=siem.config.backend if siem.config else None,
        destination=siem.config.destination if siem.config else None,
        last_checked_at=siem.last_checked_at,
        error_detail=siem.last_error if siem.consecutive_failures > 0 else None,
        consecutive_failures=siem.consecutive_failures,
        dropped_events=siem.dropped_events,
    )
```

### Task 4: Initialize SIEMService in main.py lifespan

**Files:** `puppeteer/agent_service/main.py` (MODIFIED)

**Action:** UPDATE main.py lifespan to initialize SIEMService. Follow vault_service initialization pattern (PATTERNS.md lines 23-90).

Find the lifespan function and add SIEM init block after vault init (around line ~80):

```python
# Inside lifespan startup (after vault initialization)

# Initialize SIEM service (Phase 168)
try:
    from agent_service.db import SIEMConfig
    from ee.services.siem_service import SIEMService, set_active
    
    siem_config_result = await db.execute(select(SIEMConfig).limit(1))
    siem_config = siem_config_result.scalars().first()
    
    if siem_config and siem_config.enabled:
        siem_service = SIEMService(siem_config, db, scheduler)
        await siem_service.startup()
        set_active(siem_service)
        app.state.siem_service = siem_service
        logger.info("SIEM service initialized and active")
    else:
        set_active(None)
        logger.info("SIEM service disabled or not configured")
except ImportError:
    # CE mode: SIEM not available
    set_active(None)
except Exception as e:
    logger.error(f"Failed to initialize SIEM service: {e}")
    set_active(None)
```

And in the shutdown (after vault shutdown):

```python
# Inside lifespan shutdown (after vault shutdown)

try:
    siem = app.state.siem_service if hasattr(app.state, 'siem_service') else None
    if siem:
        await siem.shutdown()
        logger.info("SIEM service shutdown complete")
except Exception as e:
    logger.error(f"Error shutting down SIEM service: {e}")
```

async def health():

        if vault:

    except Exception:
        vault_status = "degraded"
    
    siem_status = "disabled"

        siem = get_siem_service()
        if siem:
            siem_status = await siem.status()
    except Exception:

    return {
        "status": "ok",
        "vault": vault_status,
        "siem": siem_status,
    }
