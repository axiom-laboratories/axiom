# Plan 101-01: CE Tab Gating + Upgrade Panel in Admin.tsx

## Objective

Gate the six EE-only Admin tabs (`smelter`, `bom`, `matrix`, `vault`, `rollouts`, `automation`)
behind `isEnterprise` so CE users never see them. Replace those tabs with a single
`+ Enterprise` tab whose content lists all six gated features via the shared
`UpgradePlaceholder` component (or a lightweight inline panel). Confirm no blank
pages appear in CE mode — only visible route risk was these Admin tabs, which this
plan eliminates.

## Context

`useLicence().isEnterprise` is already imported and used in `Admin.tsx`.
`UpgradePlaceholder` accepts `feature` + `description` props and renders a lock icon,
heading, description, and "Learn More" CTA. The Radix `Tabs` component uses
`defaultValue="onboarding"` — no URL routing — so removed tabs cannot be deep-linked.
