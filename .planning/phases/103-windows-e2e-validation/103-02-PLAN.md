Create `/home/thomas/Development/mop_validation/scripts/run_windows_scenario.py` as the paramiko SSH helper library for all Dwight interactions. Parallel structure to `run_ce_scenario.py`.

The file must provide these public functions:

**`_connect_dwight(secrets: dict) -> paramiko.SSHClient`** (private, used by all other helpers):
- Try key-based auth first using `secrets.get("dwight_ssh_key")` as key filename path (look for the key in `/home/thomas/Development/mop_validation/` directory)
- Fall back to password auth using `secrets["dwight_password"]`
- Connect to `secrets["dwight_ip"]` with username `secrets["dwight_username"]`
- Set `AutoAddPolicy` for host key
- Connection timeout: 15 seconds

**`dwight_exec(cmd: str, timeout: int = 60, secrets: dict = None) -> tuple[str, str, int]`**:
- Opens a fresh SSH connection per call (same pattern as test_ssh.py — no persistent connection state)
- Always prefixes with `pwsh -NoProfile -NonInteractive -Command "..."` — never bare commands (Windows SSH defaults to cmd.exe)
- For commands that contain double quotes, wrap the body in single quotes or use a temp .ps1 script approach
- Returns (stdout_text, stderr_text, exit_code)
- On timeout: return ("", f"TIMEOUT after {timeout}s", 1)

**`dwight_push(local_path: str, remote_path: str, secrets: dict) -> None`**:
- Uses paramiko SFTP (`client.open_sftp()`)
- Creates parent directories on Dwight if they don't exist (use `sftp.mkdir` with exist_ok logic)
- Transfers file as binary (no line ending conversion — files from Linux are already LF)
- remote_path: Windows path string (e.g., `C:/Users/drear/axiom-workspace/compose.cold-start.yaml`) — paramiko SFTP accepts forward slashes on Windows

**`dwight_pull(remote_path: str, local_path: str, secrets: dict) -> bool`**:
- Uses paramiko SFTP to download a file from Dwight to local_path
- Returns True on success, False if file not found (catches `FileNotFoundError`)

**`wait_for_stack_dwight(timeout: int = 300, secrets: dict = None) -> bool`**:
- Polls the Dwight dashboard from the Linux host (not via SSH) using `requests.get`
- URL: `https://192.168.50.149:8443` with `verify=False` and `timeout=10`
- Returns True if HTTP 200 received within timeout seconds
- Polls every 5 seconds, prints progress dots
- Import requests at top of file

**`ensure_workspace_dwight(secrets: dict) -> str`**:
- Creates the workspace directory on Dwight via SSH: `dwight_exec("New-Item -ItemType Directory -Force -Path C:\\Users\\drear\\axiom-workspace", ...)`
- Returns the workspace path string: `"C:/Users/drear/axiom-workspace"`

**Module-level constants** (all paths as strings, easy to patch):
```python

```

**Imports:** `import paramiko, requests, sys, os, time`. Use `sys.path.insert(0, os.path.dirname(__file__))` to allow `from test_ssh import read_secrets` to work when called from any directory.
