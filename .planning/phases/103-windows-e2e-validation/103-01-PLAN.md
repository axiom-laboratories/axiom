**Gap 1 — Step 0 keypair generation: add a Windows (PowerShell) tab.**

The existing Step 0 has two tabs: `=== "Python (cryptography)"` (uses bash heredoc) and `=== "openssl"` (uses bash). Add a third tab `=== "Windows (PowerShell)"` that uses a Python script file approach (no heredoc, which doesn't work in PowerShell):

```powershell
# Save this as gen_key.py first
$script = @'
from cryptography.hazmat.primitives.asymmetric.ed25519 import Ed25519PrivateKey
from cryptography.hazmat.primitives import serialization

key = Ed25519PrivateKey.generate()

with open("signing.key", "wb") as f:
    f.write(key.private_bytes(
        serialization.Encoding.PEM,
        serialization.PrivateFormat.PKCS8,
        serialization.NoEncryption()
    ))

with open("verification.key", "wb") as f:
    f.write(key.public_key().public_bytes(
        serialization.Encoding.PEM,
        serialization.PublicFormat.SubjectPublicKeyInfo
    ))

print("Done. Upload verification.key to Axiom, keep signing.key private.")
'@
$script | Out-File -Encoding utf8 gen_key.py
python gen_key.py
```

!!! note "cryptography library"

1. Sign the script using Python (cryptography library — same library as keypair generation, avoids openssl dependency):
```powershell
# Sign hello.ps1 (or hello.py) using Python
$signScript = @'
import base64, sys
from cryptography.hazmat.primitives import serialization
from cryptography.hazmat.primitives.asymmetric.ed25519 import Ed25519PrivateKey

with open("signing.key", "rb") as f:
    key = serialization.load_pem_private_key(f.read(), password=None)

with open(sys.argv[1], "rb") as f:
    script_bytes = f.read()

sig = key.sign(script_bytes)
print(base64.b64encode(sig).decode())
'@
$signScript | Out-File -Encoding utf8 sign_script.py
$SIG = python sign_script.py hello.ps1
```

2. Read the script content and dispatch via Invoke-RestMethod:
```powershell
$scriptContent = Get-Content -Raw hello.ps1
$body = @{
    script_content = $scriptContent
    signature = $SIG
    signature_key_id = "<your-key-id>"
} | ConvertTo-Json

Invoke-RestMethod -Method POST `
    -Uri "https://<your-orchestrator>:8001/jobs" `
    -Headers @{Authorization = "Bearer $TOKEN"} `
    -ContentType "application/json" `
    -Body $body
```
