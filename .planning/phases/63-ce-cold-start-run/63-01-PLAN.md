  <action>
Create `mop_validation/scripts/run_ce_scenario.py` with the following functions. This is the run orchestrator that Plan 02 and Plan 03 will call.

**`incus_exec(cmd: str, timeout: int = 30) -> subprocess.CompletedProcess`**
- Runs `["incus", "exec", "axiom-coldstart", "--", "bash", "-c", cmd]` with capture_output=True, text=True
- Returns the CompletedProcess; does not raise on non-zero — callers check returncode

**`incus_push(local: str, container_path: str) -> None`**
- Runs `["incus", "file", "push", local, f"axiom-coldstart{container_path}"]`
- Raises subprocess.CalledProcessError on failure (use check=True)

**`incus_pull(container_path: str, local: str) -> bool`**
- Runs `["incus", "file", "pull", f"axiom-coldstart{container_path}", local]`
- Returns True if returncode == 0, False otherwise

**`wait_for_stack(timeout: int = 600) -> bool`**
- Polls every 5 seconds up to `timeout` seconds
- Each poll: `incus_exec("curl -k -s -o /dev/null -w '%{http_code}' https://172.17.0.1:8443", timeout=15)`
- Returns True when stdout.strip() in ("200", "301"), False on timeout
- Prints progress: `Attempt N: got STATUS — waiting 5s`

**`reset_stack(compose_src: str = "/home/thomas/Development/master_of_puppets/puppeteer/compose.cold-start.yaml") -> None`**
- Step 1: Push latest compose file: `incus_push(compose_src, "/workspace/compose.cold-start.yaml")`
- Step 2: Hard reset: `incus_exec("cd /workspace && docker compose -f compose.cold-start.yaml down -v 2>&1", timeout=120)`
- Step 3: Clean workspace artifacts: `incus_exec("rm -f /workspace/FRICTION-CE-*.md /workspace/checkpoint/PROMPT.md /workspace/checkpoint/RESPONSE.md")`
- Step 4: Fresh start with build: `incus_exec("cd /workspace && docker compose -f compose.cold-start.yaml up -d --build 2>&1", timeout=600)`
- Prints each step as it runs

**`run_gemini_scenario(scenario_local: str, scenario_name: str) -> None`**
- Pushes scenario file from `scenario_local` to `/workspace/{scenario_name}` inside LXC
- Prints: `Launching Gemini for scenario: {scenario_name}`
- Prints: `IMPORTANT: Start monitor_checkpoint.py in a second terminal NOW before Gemini begins`
- Waits 10 seconds (to allow operator to start monitor)
- Runs: `incus_exec(f"source /etc/environment && HOME=/root/validation-home gemini -p \"$(cat /workspace/{scenario_name})\"", timeout=3600)`
- Prints stdout/stderr as it completes

**`pull_friction(scenario_id: str, reports_dir: str = "/home/thomas/Development/mop_validation/reports") -> bool`**
- scenario_id is "INSTALL" or "OPERATOR"
- Pulls `/workspace/FRICTION-CE-{scenario_id}.md` to `{reports_dir}/FRICTION-CE-{scenario_id}.md`
- Returns result of incus_pull; prints success or failure

Add `if __name__ == "__main__":` block that runs `reset_stack()` then `wait_for_stack()` and prints the result — this is what Plan 01 Task 2 invokes directly.
