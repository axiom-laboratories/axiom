### Critical: INFRA-06 Regression

**Commit:** `ab25961` — `docs(phase-22): complete phase execution` (not a plan task commit)
**Change:** Removed `- privacy` and `- offline` from `docs/mkdocs.yml` plugins list
**Current state confirmed:** `grep "privacy\|offline" docs/mkdocs.yml` → no output

Impact:
- Any `docker compose build docs` today produces a CDN-dependent image
- Google Fonts loaded at runtime from `fonts.googleapis.com` / `fonts.gstatic.com`
- `docs/docs/security/air-gap.md` line 155 checklist item "Documentation site loads without external requests" cannot be satisfied

**Fix (2 lines in docs/mkdocs.yml):**
```yaml
plugins:
  - search
  - swagger-ui-tag
  - privacy   # ← restore
  - offline   # ← restore
```

Note: both `privacy` and `offline` are bundled with `mkdocs-material` (no additional pip package needed). The `privacy` plugin may require `pillow` and `cairosvg` system libs for image processing — check if these are available in the Docker builder stage.
