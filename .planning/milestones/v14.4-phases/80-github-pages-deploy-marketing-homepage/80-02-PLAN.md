  <name>Task 2: Create .github/workflows/homepage-deploy.yml</name>
  <files>.github/workflows/homepage-deploy.yml</files>
  <action>
Create `.github/workflows/homepage-deploy.yml` using the pattern from the existing `docs-deploy.yml` for action version consistency. Full workflow:

```yaml
name: homepage-deploy

on:
  push:
    branches: [main]
    paths:
      - 'homepage/**'
      - '.github/workflows/homepage-deploy.yml'

permissions:
  contents: write

jobs:
  deploy:
    runs-on: ubuntu-latest
    steps:
      - uses: actions/checkout@v4
        with:
          fetch-depth: 0

      - name: Configure Git Credentials
        run: |
          git config user.name github-actions[bot]
          git config user.email 41898282+github-actions[bot]@users.noreply.github.com

      - name: Deploy homepage to gh-pages root
        run: |
          git fetch origin gh-pages
          git checkout gh-pages
          cp homepage/index.html index.html
          cp homepage/style.css style.css
          touch .nojekyll
          git add index.html style.css .nojekyll
          git diff --cached --quiet || git commit -m "Deploy homepage from ${{ github.sha }}"
          git pull --rebase origin gh-pages
          git push origin gh-pages
```

CRITICAL implementation rules:
- `git diff --cached --quiet || git commit` — prevents empty-commit errors when files have not changed
- `git pull --rebase origin gh-pages` — defensive rebase before push to handle concurrent workflow edge case
- NEVER write to `docs/` in this workflow — only `index.html`, `style.css`, `.nojekyll` at root
- `fetch-depth: 0` — required so git can checkout a non-default branch
- `cp homepage/index.html index.html` copies from the `main` branch checkout (homepage/ dir is in the working tree at the start, before `git checkout gh-pages` replaces the working tree — IMPORTANT: the `cp` runs AFTER checkout, so it must reference the file from the checked-out branch. Use a different approach: store files before checkout)

CORRECTED sequence — copy files from main before switching branches:
```yaml
- name: Deploy homepage to gh-pages root
  run: |
    # Stash the source files from main branch before switching
    mkdir -p /tmp/homepage-deploy
    cp homepage/index.html /tmp/homepage-deploy/index.html
    cp homepage/style.css /tmp/homepage-deploy/style.css
    # Switch to gh-pages and deploy
    git fetch origin gh-pages
    git checkout gh-pages
    cp /tmp/homepage-deploy/index.html index.html
    cp /tmp/homepage-deploy/style.css style.css
    touch .nojekyll
    git add index.html style.css .nojekyll
    git diff --cached --quiet || git commit -m "Deploy homepage from ${{ github.sha }}"
    git pull --rebase origin gh-pages
    git push origin gh-pages
```
