```python
CONTAINER = "axiom-coldstart"
PROMPT_REMOTE = "/workspace/checkpoint/PROMPT.md"
RESPONSE_REMOTE = "/workspace/checkpoint/RESPONSE.md"
CHECK_INTERVAL = 5  # seconds (use 5 for verification test, 30 for live monitoring)

def pull_file_from_lxc(container, remote_path, local_path):
    result = subprocess.run(
        ["incus", "file", "pull", f"{container}{remote_path}", str(local_path)],
        capture_output=True,
    )
    return result.returncode == 0

def push_file_to_lxc(container, local_path, remote_path):
    subprocess.run(
        ["incus", "file", "push", str(local_path), f"{container}{remote_path}"],
        check=True,
    )
```

```python
CONTAINER = "axiom-coldstart"
PROMPT_REMOTE = "/workspace/checkpoint/PROMPT.md"
RESPONSE_REMOTE = "/workspace/checkpoint/RESPONSE.md"
DEFAULT_INTERVAL = 30   # seconds for live monitoring
VERIFY_INTERVAL = 5     # seconds for verification tests
```

**CLI arguments:**
- `--interval N` — override poll interval (default: 30 for live mode)
- `--verify-mode` — use 5-second poll interval, print timing data, exit after first round-trip detected
- `--once` — check once for a PROMPT.md and exit (for scripted testing)

**Main polling loop (`monitor_loop`):**
1. Print: "Monitoring axiom-coldstart:/workspace/checkpoint/ for PROMPT.md (interval={N}s)..."
2. Print: "Press Ctrl+C to stop."
3. Loop:
   a. `pull_file_from_lxc(CONTAINER, PROMPT_REMOTE, /tmp/checkpoint-prompt.md)` — try to pull
   b. If pull succeeds (PROMPT.md exists in LXC):
      - Print: "\n[CHECKPOINT] Gemini is blocked. PROMPT.md received at <timestamp>"
      - Print: "--- PROMPT CONTENT ---"
      - Print contents of /tmp/checkpoint-prompt.md
      - Print: "--- END PROMPT ---"
      - Ring terminal bell: `print("\a", end="", flush=True)`
      - Prompt operator: `response_text = input("\nEnter response for Gemini (or 'skip' to let it time out): ")`
      - If response_text.strip().lower() == "skip": print "Skipping — Gemini will time out on its own" and continue loop
      - Write response to /tmp/checkpoint-response.md
      - Push to LXC: `push_file_to_lxc(CONTAINER, "/tmp/checkpoint-response.md", RESPONSE_REMOTE)`
      - Print: "[CHECKPOINT] RESPONSE.md pushed to LXC at <timestamp>"
      - If `--verify-mode`: also record round-trip elapsed time and print it; exit 0 after first round-trip
   c. Sleep interval, repeat

**Helper: `check_once` mode (--once flag):**
- Try one pull; if PROMPT.md found, print contents and exit 0; if not found, print "No checkpoint pending" and exit 1

**Main guard:** `if __name__ == "__main__": main()`

The `--verify-mode` flag is specifically designed for the SCAF-02 round-trip verification test — it exits cleanly after detecting and responding to one checkpoint, making it automatable.

```python
def check_scaf02_checkpoint_roundtrip():
    """
    Verify the checkpoint file protocol round-trip in under 60 seconds.
    1. Write a synthetic PROMPT.md to the LXC
    2. Pull it back to confirm incus file pull works
    3. Write a synthetic RESPONSE.md to the LXC
    4. Confirm RESPONSE.md is readable from inside the LXC
    5. Clean up both files
    Total time tracked — must be < 60 seconds
    """
    import time
    import tempfile
    start = time.time()
    results = []

    # Step 1: Write synthetic PROMPT.md to LXC
    with tempfile.NamedTemporaryFile(mode='w', suffix='.md', delete=False) as f:
        f.write("## What I was doing\nTest step\n\n## What happened\nTest error\n\n## What I need\nTest question\n")
        prompt_local = f.name

    ok = push_file_to_lxc(CONTAINER, prompt_local, "/workspace/checkpoint/PROMPT.md")
    results.append(("Push PROMPT.md to LXC", ok))

    # Step 2: Pull PROMPT.md back to confirm pull works
    pulled = pull_file_from_lxc(CONTAINER, "/workspace/checkpoint/PROMPT.md", "/tmp/scaf02-verify-prompt.md")
    results.append(("Pull PROMPT.md from LXC", pulled))

    # Step 3: Write synthetic RESPONSE.md to LXC
    with tempfile.NamedTemporaryFile(mode='w', suffix='.md', delete=False) as f:
        f.write("Test response from Claude.\n")
        response_local = f.name

    ok = push_file_to_lxc(CONTAINER, response_local, "/workspace/checkpoint/RESPONSE.md")
    results.append(("Push RESPONSE.md to LXC", ok))

    # Step 4: Verify RESPONSE.md is readable from inside LXC
    result = subprocess.run(
        ["incus", "exec", CONTAINER, "--", "test", "-f", "/workspace/checkpoint/RESPONSE.md"],
        capture_output=True,
    )
    results.append(("RESPONSE.md readable in LXC", result.returncode == 0))

    # Step 5: Clean up
    subprocess.run(
        ["incus", "exec", CONTAINER, "--", "rm", "-f",
         "/workspace/checkpoint/PROMPT.md", "/workspace/checkpoint/RESPONSE.md"],
        capture_output=True,
    )

    elapsed = time.time() - start
    results.append((f"Total round-trip < 60 seconds (actual: {elapsed:.1f}s)", elapsed < 60))

    for label, passed in results:
        status = "PASS" if passed else "FAIL"
        print(f"  [{status}] [SCAF-02] {label}")

    return all(ok for _, ok in results)
```
