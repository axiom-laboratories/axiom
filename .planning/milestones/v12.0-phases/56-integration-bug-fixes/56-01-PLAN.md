**Helper functions:**

`load_secrets()`: Read `Path(__file__).resolve().parents[2] / "master_of_puppets" / "secrets.env"`. Parse KEY=VALUE lines (skip # lines), return dict. Extract ADMIN_PASSWORD.

`get_jwt_token(password)`: POST to `{BASE_URL}/auth/login` with `data={"username": "admin", "password": password}`, `verify=False`. Return `r.json()["access_token"]`.

`load_or_generate_signing_key()`: Load `Path(__file__).resolve().parents[2] / "master_of_puppets" / "secrets" / "signing.key"` if it exists (PEM bytes). If not, generate a new `Ed25519PrivateKey.generate()` and write PEM to that path (using `NoEncryption()`). Return the private key object.

`ensure_signing_key_registered(token, private_key)`: Export the public key as PEM (`private_key.public_key().public_bytes(Encoding.PEM, PublicFormat.SubjectPublicKeyInfo)`). Call `GET {BASE_URL}/signatures` with auth header to list existing keys. Check if any key's `public_key_pem` matches. If not found, POST to `{BASE_URL}/signatures` with `{"name": "phase56-test-key", "public_key_pem": pem_str}`. Return `sig_id` (the id from the matching or newly created signature record).

`sign_script(private_key, script_content)`: Return `base64.b64encode(private_key.sign(script_content.encode("utf-8"))).decode()`.

`submit_job(token, sig_id, sig_b64, script_content, runtime, max_retries=0, target_tags=None)`: POST to `{BASE_URL}/jobs` with the job JSON payload (use `script_content` key inside payload dict — INT-01 fix). Return `guid`.

`wait_for_job_status(token, guid, terminal_statuses=("COMPLETED","FAILED","DEAD_LETTER","CANCELLED"), timeout=90)`: Poll `GET {BASE_URL}/jobs/{guid}` every 2s until status is in terminal_statuses or timeout reached. Return final status string.

`get_job_from_list(token, guid)`: GET `{BASE_URL}/jobs`, find item with matching guid in `r.json()["items"]`, return it.
