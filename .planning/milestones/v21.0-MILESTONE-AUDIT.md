### End-to-End Flows

#### Flow 1: On-Demand Job Dispatch

```
POST /jobs (script_content)
  → Phase 131: countersign_for_node(script_content) [mandatory, hard-fail]
  → payload_dict["signature"] = server_sig
  → JobService.create_job() → DB
  → Phase 129: response_model=JobResponse serializes response
  → Phase 130: test_happy_path_dispatch validates JobResponse(**result)
  → Node pulls /work/pull → WorkResponse (Phase 129)
  → Node executes → result returned
```

**Status: COMPLETE ✓**

#### Flow 2: Scheduled Job Dispatch

```
APScheduler fires execute_scheduled_job()
  → Phase 131: countersign_for_node(s_job.script_content) [mandatory]
  → On error: fire_log.status='signing_error' + AuditLog + early return
  → payload_dict["signature"] = server_sig
  → HMAC stamp: compute_signature_hmac(ENCRYPTION_KEY, sig, s_job.id, guid)
  → Job created in DB with signature + hmac
  → Node polls → dispatched with server-signed payload
```

**Status: COMPLETE ✓**
