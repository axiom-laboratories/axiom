Locate the `create_job()` async function in `job_service.py`. At the start of the function body, before the `Job(...)` object is constructed, add an ONLINE node availability check for the given `env_tag`:

```python
# Validate that at least one ONLINE node can serve this env_tag
if env_tag:
    from sqlalchemy import select
    result = await db.execute(
        select(Node).where(
            Node.status == "ONLINE",
            Node.env_tag == env_tag,
        )
    )
    eligible = result.scalars().first()
    if eligible is None:
        raise HTTPException(
            status_code=422,
            detail={
                "error": "no_eligible_node",
                "env_tag": env_tag,
                "message": f"No ONLINE node with env_tag='{env_tag}' available.",
            },
        )
```

Use the import style already in the file (check existing imports at top of `job_service.py`). If `HTTPException` is not already imported in job_service.py, add `from fastapi import HTTPException`. If `Node` DB model is not already imported, add the import.
