Steps (in order):

2. Verify compose file exists: `[ -f "$COMPOSE_FILE" ] || { echo "[ERROR] Compose file not found: $COMPOSE_FILE"; exit 1; }`
3. Run: `docker compose -f "$COMPOSE_FILE" -p "$PROJECT_NAME" down -v --remove-orphans` — removes all named volumes
4. Print: "[HARD TEARDOWN] All Docker volumes removed."
5. Print: "[HARD TEARDOWN] Clearing LXC node secrets dirs..."
6. Discover and clear axiom-node-* containers:
```bash
while IFS=',' read -r name rest; do
    name=$(echo "$name" | tr -d '"')
    if [[ "$name" == axiom-node-* ]]; then
        echo "  Clearing $name:/home/ubuntu/secrets/ ..."
        incus exec "$name" -- rm -rf /home/ubuntu/secrets/ \
            || echo "  [WARN] Could not clear $name (not running or incus exec failed) — skipping"
    fi
done < <(incus list --format csv 2>/dev/null || true)
```
7. Print: "[HARD TEARDOWN] Complete. True clean slate — no Root CA, no node certs, no DB data."
8. Exit 0
