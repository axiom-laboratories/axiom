**load_env(path) function** — same as test_local_stack.py: reads file line by line, skips blank lines and # comments, splits on first =.

**check(name, passed, detail="") function:**
- Prints "[PASS] {name}" or "[FAIL] {name}" with optional detail
- Returns passed (bool)

**get_postgres_container() function:**
- Run: `docker ps --filter "name=puppeteer-db" --format "{{.Names}}"` via subprocess
- Strip and return first line result
- Fallback to "puppeteer-db-1" if empty or fails

**wait_for_stack(url, timeout=90) function:**
- Poll GET /url/api/health every 3 seconds until status < 500 or timeout
- Print dots during wait: `print(".", end="", flush=True)` each iteration
- Print newline on completion
- Return True if ready, False on timeout

**get_admin_token(base_url, password) function:**
- POST /auth/login with form data {username: "admin", password: password}
- verify=False, timeout=10
- Return access_token string on 200, None on failure

**check_table_count(container) function:**
- Run subprocess: docker exec {container} psql -U puppet -d puppet_db -t -c "SELECT count(*) FROM pg_tables WHERE schemaname='public' AND tablename != 'apscheduler_jobs';"
- Parse int from stdout.strip()
- Return (passed: bool, count: int) where passed = (count == CE_TABLE_COUNT)
- On subprocess error: return (False, 0)

**check_features(base_url) function:**
- GET /api/features (no auth needed — unauthenticated endpoint)
- verify=False, timeout=10
- Return False if status != 200
- Verify features.keys() == CE_FEATURE_KEYS AND not any(features.values())
- Return bool result

**check_licence_ce(base_url, token) function:**
- GET /api/licence with Authorization: Bearer {token}
- verify=False, timeout=10
- Return False if status != 200
- Return data.get("edition") == "community"

**main() function:**

```python
def main():
    print("=" * 60)
    print("Axiom CE Install Verification")
    print("Target: axiom-split CE stack at https://localhost:8001")
    print("=" * 60)

    results = []

    # Load admin password
    secrets = load_env(SECRETS_ENV)
    admin_password = secrets.get("ADMIN_PASSWORD", "")
    if not admin_password:
        print("[WARN] ADMIN_PASSWORD not found in secrets.env — trying 'admin123'")
        admin_password = "admin123"

    # Wait for stack
    print(f"\nWaiting for stack at {AGENT_URL} ", end="")
    if not wait_for_stack(AGENT_URL):
        print(f"\n[FAIL] Stack not ready after 90s — is the axiom-split CE stack running?")
        sys.exit(1)
    print("[OK] Stack is up")

    # Get postgres container name
    pg_container = get_postgres_container()
    print(f"\nPostgres container: {pg_container}")

    # INST-03 check 1: Table count
    passed, count = check_table_count(pg_container)
    results.append(check(
        "INST-03 — CE table count",
        passed,
        f"found {count} tables (expected {CE_TABLE_COUNT}, excluding apscheduler_jobs)"
    ))

    # INST-03 check 2: /api/features
    results.append(check(
        "INST-03 — /api/features all false",
        check_features(AGENT_URL),
        f"expected 8 keys all false"
    ))

    # Get admin token for auth-required checks
    token = get_admin_token(AGENT_URL, admin_password)
    if not token:
        results.append(check("Admin login", False, f"Could not log in as admin — check ADMIN_PASSWORD in secrets.env"))
    else:
        results.append(check("Admin login", True, "admin token obtained"))

        # INST-03 check 3: /api/licence
        results.append(check(
            "INST-03 — /api/licence edition == community",
            check_licence_ce(AGENT_URL, token),
            "expected {\"edition\": \"community\"}"
        ))

    # INST-04 reminder
    print("\n" + "-" * 60)
    print("INST-04: Admin re-seed safety is a MANUAL test.")
    print("See INST_04_MANUAL_TEST_STEPS in this script for steps.")
    print("-" * 60)

    # Summary
    passed_count = sum(1 for r in results if r)
    total = len(results)
    print(f"\nResult: {passed_count}/{total} checks passed")

    if passed_count == total:
        print("\n[ALL PASS] CE install verified.")
        sys.exit(0)
    else:
        print("\n[FAILURES DETECTED] Review [FAIL] lines above.")
        sys.exit(1)

if __name__ == "__main__":
    main()
```
