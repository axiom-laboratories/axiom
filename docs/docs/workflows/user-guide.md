# User Guide

Learn how to monitor and interact with workflows through the Axiom dashboard.

This guide covers:
- **Workflows List** — View all workflows and their execution history
- **WorkflowDetail** — Inspect a specific workflow and view past runs
- **WorkflowRunDetail** — Monitor live workflow execution with the DAG canvas overlay
- **Step Logs** — Inspect detailed logs for individual workflow steps
- **Gate Types** — Understanding conditional branching and parallel execution
