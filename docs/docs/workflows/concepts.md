# Concepts

Learn the fundamental building blocks of Axiom workflows.

This page covers:
- **Step Types** — Action nodes that execute scripts or commands
- **Gate Node Types** — Control flow gates: IF_GATE, AND_JOIN, OR_GATE, PARALLEL, SIGNAL_WAIT
- **DAG Model** — How steps and edges form a directed acyclic graph
- **Execution Lifecycle** — From creation through completion or cancellation
