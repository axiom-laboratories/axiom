# Workflow Documentation

This section documents workflow features. Learn how to create, monitor, and manage workflows in Axiom.

**Sections:**

- **Concepts** — Understand the fundamental building blocks: step types, gate node types, DAG model, and execution lifecycle.
- **User Guide** — Monitor workflows from the dashboard: Workflows list, WorkflowDetail, WorkflowRunDetail, and step logs.
- **Operator Guide** — Observable behavior, status transitions, and operational setup (cron/webhook configuration via API).
- **Developer Guide** — Internal implementation: BFS dispatch, concurrency guards, cascade cancellation, and data model.
