# Operator Guide

Learn how to operate and monitor workflows in Axiom at runtime.

This guide covers:
- **Status Transitions** — Understanding the 6 workflow statuses and state machine
- **Observable Behavior** — What triggers a run, how parameter injection works, cascade cancellation
- **Cron Job Setup** — Scheduling workflows with cron expressions via API
- **Webhook Configuration** — Triggering workflows with webhooks and HMAC signing
- **API Monitoring** — Using the workflow API to check status, list runs, and manage executions
